行末的省略号设置

Posted on 2017-05-12

单行文本溢出：

设置要点：

  1. 元素宽度要小于文本单行长度，
  2. 同时保持单行文本不折行
  3. 1、2共同作用造成文本溢出，显示省略符号来代表被修剪的文本
  4. 溢出文本不显示

多行文本溢出：

设置要点：

  1. 元素宽度要小于文本单行长度，出现文本折行的情况
  2. 一定要设置高度和行高。
  3. 元素高度和行高成比例，若希望省略号出现在最后一行，则 -webkit-line-clamp的值即为该比例。-webkit-line-clamp 的值n表示在第 n 行末尾产生省略号
  4. 辅助设置：display: -webkit-box; -webkit-box-orient: vertical;
  5. 仅适用于webkit内核浏览器
  6. 省略号相当于插入到句子内部，不替换句子中原来的字符
