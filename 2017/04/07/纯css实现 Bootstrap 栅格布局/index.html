Bootstrap 栅格布局实现

Veröffentlicht am 2017-04-07

这是百度前端学院前端基础部分的一个题目，使用HTML和CSS完成类似Bootstrap的12栏栅格布局

在Bootstrap官网上有关于栅格系统实现的说明。栅格系统基于flexbox构建，通过设置行和列来进行布局。

这个大问题主要涉及到：

  1. flex布局
  2. 对每一行进行切割，对列宽（在每一行中的比例）的控制
  3. 响应式效果的实现

响应式实现

响应式主要是通过媒体查询来实现的。Bootstrap提供了五个层级。

层级 超小 小 中 大 超大
宽度 <576px >=576px >=768px >=992px >=1200px
类名 .col- .col-sm- .col-md- .col-lg- .col-xl-

所以，可以通过 @media 设置四个断点: 576px, 768px, 992px, 1200px 。 再分别对带有层级前缀的类进行设置

给div添加不同的类，在不同的宽度下，div会响应到不同的宽度
