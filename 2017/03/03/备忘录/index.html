hexo 4000端口

  • 运行完hexo server给出正常提示后，在浏览器打开 http://localhost:4000
  • 然而并没有看到 Hello World，且页面显示一直在加载

  • 有一种可能的情况就是 4000端口被占用（福昕阅读器的默认端口也是4000）

  • 当然代码中的5000端口是可以任意更换的

    同步到github

  • 在github已经部署ok的情况下，以后的每次提交，都要进行清空
  • 依次运行 
    hexo clean
hexo generate
hexo deploy
